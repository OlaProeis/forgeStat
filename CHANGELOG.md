## [0.1.0] - 2025-03-01

### Added
- Initial release of forgeStat
- Real-time GitHub repository dashboard with 8 metric panels
- Repository Health Score calculation (0-100) based on activity, community, maintenance, and growth
- Interactive TUI with zoom mode, mini-map, and mouse support
- Star milestone prediction based on growth trends
- JSON export mode for programmatic access
- Summary view for quick status checks
- Markdown report generation
- Multi-repo watchlist dashboard
- Compare mode for side-by-side repository comparison
- Diff mode for snapshot comparison
- Command palette with autocomplete
- Fuzzy finder for repository switching
- Copy to clipboard functionality
- 6 built-in themes (default, monochrome, high-contrast, solarized-dark, dracula, gruvbox)
- Custom theme support via TOML configuration
- Animations with low-power mode
- Smart caching with 15-minute TTL and offline fallback
- Configurable via TOML files (theme, status bar, layout, animations, watchlist)
- Cross-platform support (Windows, Linux, macOS)
- GitHub Personal Access Token authentication support
